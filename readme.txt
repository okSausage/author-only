A WordPress plugin to restrict logged in users to only be able to view and edit their own posts.
  
== Description ==
  
This plugin limiting access to specific content such as posts and media attachments only to the author of the post or media.  It is ideal for multi author sites.
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
  
== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
To use this plugin you just install it and activate it.
  
= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
